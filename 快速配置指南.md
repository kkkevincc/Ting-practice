# 🚀 快速配置指南

## 立即配置步骤

### 1. 编辑 `backend/.env` 文件

在 `backend` 目录下，编辑或创建 `.env` 文件，添加以下内容：

```env
# ========== 音频转写API配置 ==========
# 保持你现有的 SiliconFlow API 密钥
SILICONFLOW_API_KEY=你的_siliconflow_密钥

# ========== AI关键词筛选API配置 ==========
# 新增：AI API配置（用于智能关键词筛选）
AI_API_BASE_URL=https://agentrouter.org/v1/chat/completions
AI_API_KEY=sk-gxVAvnI3eekI0lUkazwQKyqZBAucNsupC5uqKo0YqnPoxswe
AI_API_MODEL=glm-4.6
USE_AI_KEYWORD_FILTER=true

# ========== 其他配置 ==========
JWT_SECRET=your-secret-key-change-in-production
PORT=3001
```

### 2. 重启后端服务

```bash
# 停止当前后端服务（如果正在运行）
# 然后重新启动
cd backend
npm run dev
```

### 3. 验证配置

查看后端控制台输出，应该看到：

```
🤖 使用AI API智能筛选关键词...
   📡 AI API端点: https://agentrouter.org/v1/chat/completions
   🤖 使用模型: glm-4.6
   🔑 API密钥: sk-gxVAvnI3...
```

## 🔍 如果遇到问题

1. **检查环境变量**：确认 `.env` 文件中的配置正确
2. **查看后端日志**：检查是否有错误信息
3. **验证API密钥**：确认密钥有效
4. **重启服务**：修改 `.env` 后必须重启后端服务
